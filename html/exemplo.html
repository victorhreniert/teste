Fluxo Principal

  • init : iniciar repositorio

  • * Criações de arquivos e projetos

  • add : traquear arquivos para poder rastrear

  • status : verifica quais os estados que os arquivos estão

  • commit: ele altera ultimo estágio do arquivo no repositorio local antes para ir para o repositorio remoto

  • git ignore

  • cat: O comando cat ( abreviação de “concatenar“ ) O comando cat nos permite criar arquivos únicos ou múltiplos, visualizar o conteúdo de um arquivo, concatenar arquivos e redirecionar a saída no terminal ou arquivos
